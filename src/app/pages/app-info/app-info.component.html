Willkommen bei Share-it

Share-it ist eine Web-App um Ausgaben in einer Gruppe zu teilen

Die MVP Version soll folgende Features haben:

  • Account erstellen
  • Gruppen anlegen
  • Gruppen teilen und beitreten
  • Ausgaben hinzufügen

Für die fertige Version ist folgendes geplant:

  • Bilder für Gruppen hochladen
  • Bilder für Ausgaben hochladen
  • Ausgleichszahlungen berechnen
  • User direkt über einen link einladen

Share-it basiert auf Angular und nutzt Firebase als Backend
